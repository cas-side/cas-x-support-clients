# cas-x-support-clients

cas委托代理扩展客户端

## 使用

```xml
<dependency>
    <groupId>com.github.cas-side</groupId>
    <artifactId>cas-x-support-clients</artifactId>
    <version>1.0.3</version>
</dependency>
```
